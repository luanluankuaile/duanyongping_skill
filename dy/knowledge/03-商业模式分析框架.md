# 段永平论商业模式——分析框架

> 以下内容均整理自段永平原话，来源为《段永平投资问答录》及网易博客。
>
> **本文用于分析流程 Step 3 的"商业模式评估"环节。** 每个维度都包含：段永平原话 + 诊断问题 + 好/差信号清单。
> 评估时，**不是逐条对照打分**，而是整体判断：这个维度的"确定性是高/中/低"？

---

## 一、总论：商业模式是分析的起点

> **"首先看生意模式！这句话可能值一辈子的午餐。"**

> **"投资最重要的还是要看生意模式，不然的话大家就很容易只见树木不见森林，全部盯着眼前一个月一个季度的波动来决定买卖的行为，这样是很难做投资的。"**

### 什么是生意模式？

> **"生意模式就是赚钱的模式，最后赚得越多的模式就应该越好。"**

> **"说不清楚什么是生意模式没关系，但要知道它到底是什么意思。"**

段永平给了一个**实践主义的定义**——不是学术定义，而是"能赚钱的模式就是好模式"。如何判断？用案例对比来感受：

> **"如果还不明白什么叫生意模式，那就看看喜诗糖果，看看可口可乐、看看比亚迪、看看苹果、看看航空公司、看看那些做太阳能晶片的'光伏企业'......"**

好模式的代表：喜诗糖果、可口可乐、苹果、比亚迪（当时的判断）
差模式的代表：航空公司、光伏企业

### 商业模式在决策框架中的位置

```
┌─────────────────────────────────────┐
│ 第一关：商业模式 ← 你在这里           │
│ 权重：40%（最高，有一票否决权）        │
├─────────────────────────────────────┤
│ 核心问题：这个生意天生好吗？           │
│ 如果不好，好的管理层也救不了它         │
│ 如果好，但看不清楚，也不能碰           │
└─────────────────────────────────────┘
```

## 二、护城河

> **"护城河实际上我觉得是生意模式中的一部分，好的生意模式往往具有很宽的护城河。"**

### 段永平案例中的护城河类型

从段永平分析过的公司中，可以归纳出六种他认可的护城河：

| 护城河类型 | 段永平案例 | 核心逻辑 |
|-----------|-----------|---------|
| ① 品牌溢价 | 茅台、可口可乐 | 消费者愿意为品牌支付更高价格 |
| ② 生态系统锁定 | 苹果 | 数据、应用、硬件形成闭环，迁移成本极高 |
| ③ 网络效应 | 腾讯（微信） | 用户越多价值越大，后来者难以追赶 |
| ④ 规模+成本优势 | （段永平较少提及此类） | 规模效应降低成本 |
| ⑤ 技术/专利壁垒 | （段永平提及不多） | 别人做不了或需要付费 |
| ⑥ 转换成本 | 苹果、企业软件 | 用户切换的成本极高，不会轻易更换 |

### 段永平最看重的护城河类型

从实际案例分析，段永平最重视的是**品牌溢价**和**生态系统锁定**：

> **"苹果的平台建立起来了，或者说生意模式或者说护城河已经形成了（光软件一年都几十亿的收入了）。"**

> **"苹果厉害的不仅仅是硬件，这是所有其他竞争对手没辙的地方。三星顶多不过是新的诺基亚而已，对苹果没有威胁。"**

> **"不同白酒口感不一样，有用户粘性。喜欢喝茅台的，不会因为茅台涨价而买五粮液。"**

### 诊断问题

```
□ 护城河是什么类型的？是一个还是多个？
□ 竞争对手复制这个护城河需要多久？多难？
□ 10年后护城河会越来越宽还是越来越窄？
□ 护城河是"消费者想走但走不了"还是"消费者不想走"？

好护城河信号：
• 用户主动选择且不愿意离开
• 竞争对手花了大量资源也无法突破
• 护城河随时间会自我强化

差护城河信号：
• 主要靠低价竞争
• 用户几乎没有转换成本
• 护城河依赖某个不可复制的历史条件（如政策保护）
```

## 三、差异化与定价权

这是段永平商业模式分析中**最核心的子维度**。

> **"价格战一般在产品差异化很小时容易发生。酒是有很大差异化的东西，至少感觉如此。See's candy 也是有很大差异化的产品。"**

> **"电子消费品，尤其是成熟的电子消费品的趋势往往是差异化越来越小，尤其在专利保护比较弱的国家里会更明显。"**

### 定价权：段永平最看重的指标

> **"（迪士尼）有巨大的未使用的定价权却没在用。这是最不需要动脑筋的事。"**

段永平引用芒格：

> **"有些生意，你一生中会发现几回，任何管理者只需要通过提价就能大幅提升回报——但他们却没有这么做。所以它们有巨大的未开发的定价权却没在用。"**

### 段永平的"定价权测试"

段永平用一句话证明苹果比可口可乐好，这个测试本身就是**对他自己的定价权理论最精妙的操作化**：

> **"你看百事可乐价钱只有可口可乐的一半时，你就喝百事了。但Android和iPhone比价钱差了多少？你看真正的苹果用户没有人在乎。"**

这个测试可以分为三个层级：

| 层级 | 描述 | 案例 |
|------|------|------|
| **弱定价权** | 对手降价10%，客户大量流失 | 航空公司、商品化制造业 |
| **中定价权** | 对手降价30%，部分客户流失 | 可口可乐（百事便宜一半时巴菲特本人也会换） |
| **强定价权** | 对手降价50%以上，核心客户不流失 | 苹果（用户不在乎）、茅台（高端白酒不降价） |

### 差异化来源

段永平认可的差异化类型：

| 差异化来源 | 具体表现 | 案例 |
|-----------|---------|------|
| 产品体验 | "把用户体验或消费者导向做到极致了" | 苹果 |
| 品牌心智 | "喜欢喝茅台的，不会因为茅台涨价而买五粮液" | 茅台、可口可乐 |
| 生态系统 | 软硬件一体化、数据锁定 | 苹果 |
| 内容/社区 | 用户创造的内容和社交关系 | 腾讯（微信） |

### 关于"性价比"——红旗信号

> **"说追求'性价比'的公司大多是在为自己的低价找借口。长寿公司大概是不太强调'性价比'这个概念的，老百姓心里有杆秤。好货不便宜啊。"**

**如果一个公司把"性价比"作为核心卖点，在段永平的框架里是一个值得警惕的信号。**

### 诊断问题

```
□ 公司敢涨价吗？过去3年提价了多少次？
□ 如果竞争对手降价50%，核心客户会跑吗？（段永平测试法）
□ 产品的差异化是真实可感的，还是营销概念？
□ 用户是因为"便宜"还是因为"喜欢"而购买？

强定价权信号：
• 提价后销量不降或降幅远小于提价幅度
• 用户抱怨贵但继续买
• 竞争对手跟不跟对你无所谓

弱定价权信号：
• 靠降价促销维持销量
• 产品同质化严重，主要靠渠道或价格竞争
• 大客户议价能力极强
```

## 四、利润质量

段永平没有专门讨论"利润质量"这个术语，但他多次从不同角度触及这个问题。

### 预收款——最好的利润信号

> **"如果人们明白大部分生意实际上是没啥预收款的时候，人们会很喜欢茅台的。"**

这实际上是在说：**利润质量要看现金是不是真的到手了**。

| 利润质量 | 特征 | 案例 |
|---------|------|------|
| 高 | 预收款/先付款后交货 | 茅台（经销商先打款） |
| 中 | 正常应收，回款周期短 | 大多数消费品 |
| 低 | 大量应收账款、账期长 | 工程类、2B项目制 |
| 极低 | 利润是"算出来"的不是"收回来"的 | 部分亏损靠融资维持的公司 |

### ROE（净资产收益率）

> **"我也喜欢ROE这个指标。一般来说，ROE高的公司商业模式好。其实我想说的是ROA，因为ROE好像没包括债务？"**

> **"Roe或Roa在了解一个公司的生意模式时是有帮助的，但不是充分条件。"**

> **"'净资产收益率'是个很好的指标，可以用来排除那些你不喜欢的公司，但不能作为'核心指标'去决定你是否要投的公司。这个指标如果好的话，你首先要看的是债务；如果没有债务问题则马上要看文化，看这种获利是否可以持续。"**

**段永平的ROE使用指南：**

```
ROE高 → 要注意 → 看债务 → 
  有高负债 → ROE可能是杠杆堆出来的，不是真有那么好的生意
  无高负债 → ROE反映了真实盈利能力 → 再问文化能不能持续
```

### 诊断问题

```
□ 利润是真金白银的现金，还是账面数字？
□ 有大量预收款吗？（好信号）
□ 有大量应收账款吗？（坏信号）
□ ROE高是因为生意好，还是因为负债高？
□ 资本支出大吗？（重资产模式需要持续投入）
□ 现金流和利润匹配吗？

好利润质量信号：
• 预收款或现款现货
• 低应收账款
• 低资本支出（轻资产）
• ROE高且负债低
• 自由现金流大于净利润

差利润质量信号：
• 大量应收账款且账期越来越长
• 净利润增长但经营性现金流下降
• 高ROE伴随高负债
• 需要持续大量资本投入才能维持增长
```

## 五、长坡厚雪

> **"长长的坡，厚厚的雪！"**

这是段永平引自巴菲特的一个隐喻：

| 比喻 | 含义 | 诊断问题 |
|------|------|---------|
| 长长的坡 | 行业空间够大，增长持续 | 行业天花板多远？市场空间还在扩大吗？ |
| 厚厚的雪 | 企业有护城河，能黏住利润 | 利润率够高吗？护城河够宽吗？ |

段永平自己的解读：

> **"这里长长的坡其实不光指的是行业，而是包括企业本身能否长跑，所以企业文化很重要。"**

所以"长坡厚雪"不是纯行业分析——**行业好+企业能跑=真长坡**。

### 段永平的"长坡厚雪"案例

> **"什么公司是在长长的坡上滚着厚厚的雪呢？苹果是吧？苹果是在一个长长的坡上，似乎雪也是厚厚的。茅台当然是吧？！长长的坡，厚厚的雪，虽然偶尔会损失一点点雪。网易也应该算吧？游戏做成这个样子？！腾讯应该也算吧？Google 应该也算吧？绝对是长长的坡厚厚的雪。"**

### 诊断问题

```
□ 这个行业10年后是更大还是更小？
□ 这家公司有能力跑完全程吗？（联系企业文化）
□ 目前的市场渗透率还有多大空间？
□ 技术变革会不会让这个坡消失？（如功能手机被智能手机取代）

长坡厚雪信号：
• 行业天花板极高或仍在扩大
• 公司在行业中拥有不可替代的地位
• 好的商业模式+好的企业文化能支持长期发展
```

## 六、好生意 vs 差生意的对比框架

段永平没有列过这个清单，但从他的案例和评论中可以归纳出以下对比：

| 维度 | 好生意（可以长期持有） | 差生意（值得警惕） |
|------|---------------------|-----------------|
| 护城河 | 宽且持续加深 | 窄或正在变窄 |
| 差异化 | 显著且用户感知 | 几乎无差异化 |
| 定价权 | 强，用户对价格不敏感 | 弱，靠价格战竞争 |
| 利润质量 | 预收款、高毛利、自由现金流充裕 | 应收账款多、低毛利、重资产 |
| 竞争格局 | 寡头或垄断，新进入者难以挑战 | 高度分散，进入门槛低 |
| 客户锁定 | 转换成本高 | 转换成本低 |
| 价格敏感性 | "追求性价比"不是核心竞争力 | "性价比"是主要卖点 |

### 一票否决信号（商业模式层面）

遇到以下任何一个信号，**可以直接跳过这个公司**：

1. **产品严重同质化**——"价格战一般在产品差异化很小时容易发生"
2. **没有定价权**——竞争对手降价你就丢客户
3. **"性价比"是核心卖点**——"说追求'性价比'的公司大多是在为自己的低价找借口"
4. **高负债+高ROE**——利润可能是杠杆堆出来的
5. **重资产+低毛利**——赚的钱都要投回去维持生意
6. **行业空间在缩小**——坡在变短

## 七、商业模式案例速查

| 公司 | 段永平评价 | 商业模式判断 | 关键依据 |
|------|-----------|-------------|---------|
| **茅台** | 顶级 | 品牌护城河+提价权+预收款+低资本支出 | "如果人们明白大部分生意实际上是没啥预收款的时候，人们会很喜欢茅台的。" |
| **苹果** | 顶级 | 生态系统锁定+最强定价权+用户零转换意愿 | "苹果的生意模式实在太强大" |
| **腾讯** | 优秀 | 网络效应（微信）+大量流量 | "流量经常是很值钱的" |
| **网易** | 优秀 | 游戏行业+段永平自己了解 | "这个市场非常大"（能力圈内的判断） |
| **可口可乐** | 优秀但有替代性 | 品牌护城河但定价权弱于苹果 | 百事便宜一半时用户会换 |
| **拼多多** | 有待观察 | 流量大+本分文化 | "流量经常是很值钱的" |
| **航空公司** | 差 | 同质化+价格战+高资本支出 | 作为反面案例多次出现 |
| **光伏企业** | 差 | 同质化+没有护城河 | 作为反面案例出现 |

## 八、商业模式分析的常见错误

1. **把"好行业"等同于"好生意"**——行业好不代表具体公司能赚钱（光伏行业大但企业普遍亏损）
2. **把"现在的优势"等同于"护城河"**——现在赚钱不意味着10年后还能赚钱
3. **忽略差异化的方向**——差异化是"消费者在乎的差异"，不是"公司自己宣称的差异"
4. **只看毛利率不看利润质量**——高毛利但大量应收账款的生意仍然是差生意
5. **低估技术变革的风险**——"电子消费品的趋势往往是差异化越来越小"
