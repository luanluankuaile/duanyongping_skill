# 段永平核心原语

> 以下内容均整理自段永平原话，来源为《段永平投资问答录（投资逻辑篇）》及网易博客。
> 
> 本文收录的是**段永平投资哲学中不可再分的基础原则**，是整个知识体系的公理。
> 所有其他知识文件（商业模式、企业文化、估值）都是这些原语在具体维度的展开。
> 在分析中遇到不确定的问题时，回到这里问自己："段永平的原则对此怎么说？"

---

## 原语一：买股票就是买公司

> **"买股票就是买公司，买公司就是买公司的未来现金流折现，句号！"**

这是段永平投资哲学的**第一块基石**，整个体系都建立在这个之上。

### 展开

| 推论 | 原话 |
|------|------|
| 股票不是筹码 | "公司价值是什么？就是公司未来现金流的折现啊。未来是多久？就是直到永远的意思。" |
| 价格 vs 价值 | "记得马克思说过：价格围绕价值上下波动。买股票花的是价格，买的是价值。" |
| 简单但绝不容易 | "简单不？对，简单但绝不容易！不懂企业的人绝对没办法看懂未来现金流。" |
| 毛估估 | "所谓懂的人往往也只能看懂自己能力圈内的很少很少企业的未来现金流折现，而且还一定是毛估估滴。" |
| 不是公式 | "未来现金流折现只是一种思维方式，只有在自己能力圈范围内的公司，投资人才能毛估估看明白。没人真用公式的。" |

### 分析中的应用

- **买股票的决策 = 买下整个公司的决策**。如果决定买100股，你要问自己：如果我把整个公司都买下来，我愿不愿意？
- **你买的不是代码、不是K线、不是题材，是未来的现金**。
- **任何脱离"未来现金流"的投资理由都是投机**。

> **"其实区分所谓是不是'价值投资'的最重要，也许是唯一的点就是在'投资者'是不是在买未来现金流（的折现）。事实上的确见到很多人买股票时的理由很多时候都和未来现金流无关，但却和别的东西有关，比如市场怎么看，比如打新股一定赚钱，比如重组的概念，比如电视里那些个分析员天天在讲的那些东西。"**

## 原语二：做对的事情，把事情做对

> **"做对的事情，把事情做对！"**

这是段永平整个商业和投资哲学的**方法论核心**。

### 做对的事情（道）

"道"是方向性的选择——**选对了方向，时间是你的朋友**。

| 原则 | 原话 |
|------|------|
| 知道什么不对 | "其实，人们往往知道什么是错的事情，只要把错的事情停止做了，就离'做对的事情'更近了一步。" |
| 错了马上改 | "发现是错的事情的时候要马上停止，不管多大的代价都是最小的代价。" |
| Stop Doing List | "好的公司都一定是有一个长长的'Stop doing list'，就是'不做的事情'。" |
| 坚持正确的事 | "人们常说的'坚持到底'，指的是坚持'做对的事情'而不是坚持做错的事情！" |
| 不对的事的表述 | "什么是对的事情……往往是用什么是不对的事情来表述的。比如，骗人是不对的等等。" |

### 把事情做对（术）

"术"是执行效率——**方向对了，剩下的只是进步快慢的问题**。

| 原则 | 原话 |
|------|------|
| 术要练 | "道可学，术要练。'道'指的就是'做对的事情'……'术'指的是技术，对每个人来说都是需要学同时也需要练习的。" |
| 先道后术 | "先有道然后术才有用哈，不然就是瞎忙。" |
| 错误不可避免 | "在把事情做对的过程当中，是没有办法避免犯错误的。" |
| 每天进步一点 | "在坚持做对的事情的前提下，把事情做对的能力是可以学习的，只要每天进步一点就行。" |

### 两种错误的性质（极其重要）

> **"大家一定要分清做错的事情和在把事情做对的过程中所犯的错误的本质区别。"**

| 错误类型 | 例子 | 处理方式 |
|---------|------|---------|
| ① 做了错的事情 | 做空、借钱投资、买不懂的公司 | 马上停止，加入Stop Doing List，**不再犯** |
| ② 把事情做对中犯错 | 看错了一个季度的数据、低估了竞争 | 学习改进，"通过练习降低犯错误概率" |

**投资分析中的应用**：
- 发现公司有财务造假嫌疑 → **①类错误，直接退出**
- 预测利润偏差了20% → **②类错误，复盘学习，不一定是判断错了方向**

## 原语三：能力圈

> **"知道自己的能力圈有多大往往比自己能力圈有多大要重要的多！"**

这是段永平对"风险"的核心理解——**风险不是波动，是超出能力圈**。

| 原则 | 原话 |
|------|------|
| 能力圈边界 | "不要轻易去'扩大'自己的能力圈。搞懂一个生意往往是需要很多年的。" |
| 能力圈内很确定 | "在自己能力圈内的生意自己往往容易懂的多，对别人的不确定性对自己往往是很确定的。" |
| 懂的标准 | "看懂一家公司不会比读一个本科更容易。" |
| 筛选方法 | "我在老巴那里学到的非常重要的一点就是先看商业模式，除非你喜欢这家公司的商业模式，不然就不再往下看了。" |
| 放过是常态 | "我自己能看懂的公司也是非常少的，看不懂当然只能放弃了。" |

### 能力圈的应用

**什么是"懂"一个公司？**
> **"所谓能看懂公司就是能看懂其未来现金流。"**

具体判断标准：
- 能否说清楚**10年后**这个公司能赚多少钱（毛估估，不要精确数字）
- 能否说清楚这个生意**为什么别人做不了**
- 能否说清楚这个生意的**最大风险是什么**
- 如果股价跌了一半，是**想加仓还是想跑**

**什么不是"懂"？**
- 看过几篇研报 ≠ 懂
- 用过它的产品 ≠ 懂
- 认识它的管理层 ≠ 懂
- 听说它好 ≠ 懂

> **"有人问如何避免以为看懂实际又错了的问题，个人观点：错误是不可避免的，但呆在能力圈内以及专注和用功可以大幅度减少犯错的机会。"**

## 原语四：投资的信仰

> **"投资的信仰指的是：相信长期而言股市是称重机，对没有信仰的人而言，股市永远是投票器。"**

| 原则 | 原话 |
|------|------|
| 长期称重 | "我说的信仰就是你骨子里真的相信的东西，是不会被某些事情动摇的。比如长期利润及净现金流好的公司股价早晚会跟上来等等。" |
| 做对的事情有好结果 | "要相信的是：做对的事情最后会有好结果，或者说做错的事情最后会有大麻烦。" |
| 价值投资是唯一的投资 | "其实投资就是价值投资的意思，不然投资投的是啥？" |

### 信仰的含义

段永平说的"信仰"不是宗教意义上的，而是**不证自明的前提假设**：
- 长期来看，好公司会涨，差公司会跌
- 做对的事情最终会有好结果
- 价值投资是唯一的投资方式

**没有这个信仰，遇到波动就会动摇，无法长期持有。**

## 原语五：投资 vs 投机

> **"投机投的是零和游戏，投资投的是企业带来的利润。"**

| 维度 | 投资 | 投机 |
|------|------|------|
| 获利来源 | 企业创造的利润 | 别人的亏损（零和） |
| 关注点 | 生意、现金流、护城河 | 价格、趋势、消息、题材 |
| 时间视角 | 10年 | 天/周/月 |
| 能力要求 | 看懂生意 | "聪明"就够了 |
| 结果 | "不需要那么聪明" | "就算很聪明早晚还是会掉坑里" |
| 长期 | 剩者 | 常赌必输 |

> **"变现容易，进出方便，是绝大多数人投资亏损的首因。"**

> **"一般而言，超出一定量以后，投的企业越多赚的越少。"**

## 原语六：机会成本是唯一标尺

> **"我其实并不在乎别人这一时期赚了多少钱，我只考虑自己的机会成本。"**

这是段永平做**所有决策的实际标准**。

| 应用 | 原话 |
|------|------|
| 买什么 | "我一般只是想象如果某个公司是个非上市企业，我用目前的市值去拥有这家公司和我其他的机会比较，哪个在未来10年或20年得到的可能回报更高。" |
| 卖不卖 | "发现买错了股票或公司就应该赶紧离开，不然越到后面损失会越大。但如果懂沉没成本以及机会成本的概念的话就会容易很多。" |
| 怎么比 | "折现率实际上是相对于投资人的机会成本而言的。最低的机会成本就是无风险回报率，比如美国国债的利率。" |

### 四个机会成本的层次

```
第一层：国债/存款（无风险回报） → 基准线，投资的最低要求
    ↓
第二层：指数基金（市场平均回报） → 跑不赢指数就不如买指数
    ↓
第三层：自己看好的其他股票 → 买A还是买B？
    ↓
第四层：你能力圈内最好的那个公司 → 终极标尺
```

段永平的实际做法：他总是拿目标公司和**茅台（或苹果）**比较——如果不如茅台，那为什么要买？

> **"A股里我从若干年前就基本只拿着茅台，脑子里总是想着10年后茅台会怎么样，所以也容易拿得住。"**

## 七条原语的相互关系

```
                  ┌───────────────────┐
                  │ ① 买股票=买公司    │ ← 起点，一切从这里开始
                  │ （买未来现金流折现） │
                  └────────┬──────────┘
                           │
                  ┌────────▼──────────┐
                  │ ② 做对的事情        │ ← 方法论核心
                  │    把事情做对       │
                  └────────┬──────────┘
                           │
         ┌─────────────────┼─────────────────┐
         │                 │                 │
  ┌──────▼──────┐  ┌──────▼──────┐  ┌──────▼──────┐
  │ ③ 能力圈     │  │ ④ 投资信仰   │  │ ⑤ 投资非投机│
  │ (风险边界)   │  │ (长期称重机)  │  │ (本质区别)   │
  └──────┬──────┘  └──────┬──────┘  └──────┬──────┘
         │                 │                 │
         └─────────────────┼─────────────────┘
                           │
                  ┌────────▼──────────┐
                  │ ⑥ 机会成本        │ ← 实际决策标尺
                  │ （跟什么比？）     │
                  └───────────────────┘
```

**分析时如何回到原语：**
- 不确定某个信息是否重要 → 回到①：它影响未来现金流吗？
- 发现公司有可疑行为 → 回到②：这在"做对的事情"吗？
- 自我怀疑是否看懂 → 回到③：我在能力圈内吗？
- 股价波动想卖 → 回到④：我的信仰动摇了吗？
- 分析陷入细节 → 回到⑤：我是在投资还是投机？
- 不知道怎么选 → 回到⑥：机会成本怎么说？
