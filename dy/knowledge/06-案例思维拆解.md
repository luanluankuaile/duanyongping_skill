# 段永平案例思维拆解

> 本文件与传统的"案例整理"有本质区别：
> 
> **不是**——段永平对每个公司说过什么（罗列观点）
> **而是**——段永平在分析每个公司时，**大脑里经历了怎样的思考过程**（拆解思维）
>
> 每个案例都按照`01-认知金字塔-总纲.md`中的分析流程（排除测试 → 生意叙事 → 三维分析 → 综合判断 → 10年视角）进行还原和拆解。
> 目的是让你看到一个**思维模板**：当你自己分析公司时，应该怎么想。

---

## 案例一：苹果 vs 可口可乐——定价权的层级

### 背景

段永平在2025年浙大见面会上回忆：他与巴菲特共进晚餐时，说苹果的商业模式比可口可乐更好。巴菲特听完后说 "I got it!"，后来在伯克希尔股东大会上向比尔·盖茨介绍段永平时说："这是Ping，他告诉我苹果的商业模式比可口可乐好，所以我买了很多苹果股票。"

### 段永平的原话

> **"你看百事可乐价钱只有可口可乐的一半时，你就喝百事了。但Android和iPhone比价钱差了多少？你看真正的苹果用户没有人在乎——至少在美国是这样，在中国也一样。"**

### 思维过程拆解

**段永平的大脑里发生了什么？**

```
步骤1：在脑海中并排比较两个公司
  可口可乐 ←→ 苹果
  都是顶级消费品公司，都拥有强大的品牌护城河

步骤2：找到差异化的"关键测试"
  段永平没有列清单对比，而是设计了一个测试：
  "把价格作为变量，看消费者的行为变化"

步骤3：执行测试
  可口可乐：当百事便宜一半时，消费者（甚至巴菲特）也会换 → 护城河有裂缝
  苹果    ：当Android便宜一半甚至更多时，核心用户不换 → 护城河没有裂缝

步骤4：得出本质差异
  可口可乐的护城河 = 品牌偏好（可被价格击穿）
  苹果的护城河     = 生态系统锁定（价格无法击穿）

步骤5：结论
  苹果的商业模式 > 可口可乐的商业模式
```

### 映射到分析框架

| 框架维度 | 在这个案例中的应用 |
|---------|-----------------|
| **第一关：商业模式** | 两者都通过，但段永平进一步区分了**同一关内的层级** |
| **差异化** | 可口可乐的差异化有替代品，苹果的差异化没有近似替代品 |
| **定价权** | 可口可乐有**有限定价权**，苹果有**极强定价权** |
| **用户转换成本** | 可口可乐低（放下拿起另一瓶），苹果极高（数据/应用/硬件闭环） |
| **决策效率** | 一句话定位核心差异，不需要逐项分析——**这就是"看懂生意"** |

### 模仿要点——你分析公司时可以问的问题

> **"如果用价格作为测试变量，这个公司的核心客户会跑吗？"**

这个问题的威力在于：它一次性测试了护城河、差异化、定价权、用户粘性四个维度。

## 案例二：网易——能力圈的力量

### 背景

段永平在2002年左右以0.8美元的平均价买入网易，持有到2007年左右卖出，获得了超过100倍的回报。

### 段永平的原话

> **"当年我投网易时，市场不看好的原因是很多人觉得游戏这个市场不是很大。而我自己由于在这个行业里时间很长，所以很确定这个市场非常大（但也不知道到底有多大，事实上最后的结果比我看到的还要大）。"**

> **"这就是能力圈的意思。"**

### 思维过程拆解

```
市场共识："游戏市场不大" 

段永平的思考：
  市场的逻辑：游戏是小众娱乐，市场规模有限 → 结论：网易没有增长空间
  段永平的逻辑：我在这个行业做了很多年（小霸王、步步高），
                我亲眼看到游戏市场的爆发潜力，
                市场说的"不大"和我看到的"非常大"之间，
                我选择相信自己的判断。
                → 结论：市场错了

关键点：段永平的"懂"不是分析出来的，是"做"出来的。
  他做游戏相关生意多年，积累了对手没有的行业洞察。
  这就是能力圈的真实含义——不是"研究了多少"，而是"亲自参与了多久"。
```

### 能力圈的三个层次（从这个案例看）

| 层次 | 含义 | 在网易案例中的体现 |
|------|------|-----------------|
| 第一层：信息优势 | 你比别人知道更多信息 | 段永平知道游戏行业的实际用户数据和付费习惯 |
| 第二层：判断优势 | 你知道信息意味着什么 | 他知道"用户增长趋势"意味着"利润增长潜力" |
| 第三层：反共识勇气 | 你敢于在别人都错的时候下注 | 市场说"小"，他说"大"，并且敢重仓投入 |

### 映射到分析框架

| 框架维度 | 应用 |
|---------|------|
| **排除测试** | 段永平通过了：他懂这个生意（能力圈内）、商业模式是好的（游戏有差异化） |
| **生意叙事** | "网易是一个游戏公司，通过卖虚拟道具赚钱。我做了多年游戏，知道这个市场潜力巨大。" → 清晰简洁 |
| **安全边际=理解度** | 段永平的安全边际不是价格（0.8美元已经很低），而是他对行业的深刻理解 |
| **10年视角** | 他看到了游戏市场在未来10年的爆发趋势 |
| **反事实检验** | "如果网易跌到0.4美元，我会买更多还是想跑？" → 段永平的选择一定是买更多 |

### 模仿要点——你分析公司时可以问的问题

> **"这个公司所在的行业，我有多少第一手经验？我是在'分析'它，还是'理解'它？"**

段永平投网易不是因为"仔细分析了财务数据"，而是因为"我做了多年游戏，我知道这个行业是怎么回事"。**真正的能力圈来自亲身参与，不只是阅读和计算。**

## 案例三：茅台——一眼就能看出的好生意

### 段永平的原话

> **"如果人们明白大部分生意实际上是没啥预收款的时候，人们会很喜欢茅台的。"**

> **"我基本上只拿着茅台"——从若干年前就如此。**

> **"A股里我从若干年前就基本只拿着茅台，脑子里总是想着10年后茅台会怎么样，所以也容易拿得住。"**

### 思维过程拆解

```
段永平的逻辑链条：

第一步：看商业模式
  茅台有没有护城河？ → 有，品牌护城河（"喜欢喝茅台的，不会因为茅台涨价而买五粮液"）
  有没有定价权？     → 有，且极强（高端白酒不降价，越涨价越买）
  利润质量如何？     → 极好，预收款模式（经销商先打款后拿酒）
  轻资产还是重资产？ → 轻资产（酿酒设备相对简单，核心是品牌和工艺）

第二步：看企业——简化到一句话
  段永平用了"预收款"这个单一指标就判断了茅台生意的质量。
  不需要复杂的财务分析，一个指标就说明了一切。

第三步：10年视角
  "脑子里总是想着10年后茅台会怎么样"
  → 10年后茅台还会是中国高端白酒第一品牌
  → 10年后喝茅台的人不会变少
  → 10年后茅台赚的钱会比现在更多

第四步：机会成本
  "A股里从若干年前就基本只拿着茅台"
  → 在A股，没有比茅台更好的选择
```

### 为什么段永平可以"只看一个指标"？

段永平在这里展示了一个高级技巧：**用一个指标代表整个商业模式的判断。**

预收款代表的不只是现金质量，而是：
- 产品供不应求（需求 > 供给）
- 渠道商愿意先给钱（对品牌的信任）
- 对上下游有议价权（不依赖别人）
- 利润是"真钱"不是"账面数字"

如果你找到一个这样的**代表性格指标**，说明你真的懂了。

### 模仿要点——你分析公司时可以问的问题

> **"这个生意的核心指标是什么？用一个数字能不能说明它的质量？"**

茅台的"预收款"、苹果的"用户留存率"、腾讯的"微信DAU"——每个好生意都有一个代表性的指标。

## 案例四：腾讯——长坡厚雪的判断

### 段永平的原话

> **"长长的坡，厚厚的雪！最近确实已经投了一些了，但比例还很小。用苹果换似乎有点不舍得，我似乎更懂苹果些。"**

> **"我认为腾讯这家公司大概率10-15年内一年可以赚2000-3000亿人民币，愿意出多少钱买这家公司完全取决于你自己的机会成本。"**

> **"马化腾人不错，而且年轻，微信及微信支付的影响非常大且深远。"**

### 思维过程拆解

```
段永平的逻辑链条：

第一步：商业模式判断（长坡厚雪）
  长坡：微信的覆盖面还在扩大（年长的、海外的）、社交+支付+内容
  厚雪：微信的护城河极宽（网络效应——朋友都在上面）
  → 这是一个好生意

第二步：企业文化判断
  "马化腾人不错，而且年轻"
  → CEO本分、有长期视野、还有很长的任期（年轻）
  → 文化评估正面

第三步：10年盈利方向的毛估估
  "大概率10-15年内一年可以赚2000-3000亿"
  → 这不是精确预测，而是方向性判断
  → 基于对微信护城河的信心

第四步：机会成本比较
  "用苹果换似乎有点不舍得，我似乎更懂苹果些"
  → 注意这个比较：段永平在"腾讯 vs 苹果"之间选择
  → 两者都是好公司，但他更懂苹果（能力圈）
  → 所以在资金分配上，苹果优先

第五步：结论
  "愿意出多少钱买这家公司完全取决于你自己的机会成本"
  → 不是"腾讯好不好"，而是"对你来说腾讯是不是最好的选择"
```

### 关键洞察：段永平投了腾讯但"比例还很小"

很多人不理解：既然看好，为什么不多买？

段永平的逻辑是机会成本——不是腾讯不够好，而是**苹果更好。**
他只有一笔钱，投了腾讯就不能投苹果，而他对苹果的确定性更高。

**这在分析中是一个极其重要的原则：好坏是相对的，不是绝对的。**

### 映射到分析框架

| 框架维度 | 应用 |
|---------|------|
| **第一关：商业模式** | 腾讯通过，但可能需要更多时间验证（"要成为苹果还需要很多东西"） |
| **第二关：企业文化** | 正面（马化腾人不错、年轻），但不是段永平最熟悉的类型 |
| **第三关：价格** | 腾讯当时的价格合理，但段永平选择了机会成本更优的苹果 |
| **10年视角** | 对10-15年盈利做出了方向性判断（2000-3000亿） |
| **最终决策** | 投了，但小比例——因为比较对象（苹果）的确定性更高 |

### 模仿要点——你分析公司时可以问的问题

> **"这个公司和我能力圈内最好的公司比，哪个更好？"**

每个投资决策都是**相对选择**。不要只问"这家公司好不好"，还要问"有没有更好的"。

## 案例五：Facebook——因为不懂所以卖

### 段永平的原话

> **"FB也是家不错的公司，生意模式和文化都不错，扎克伯格年轻有为，还能好好干很多年。FB准备卖掉了，有些不舒服的东西。由于自己不用FB的任何产品，所以对FB的理解一直没达到很透彻的地步。最近一段时间，抖音就把他吓成这个样子，说明他可能还是有弱点的。我的卖出也符合'对自己不够了解的公司，涨了也想卖跌了也想卖'的情况。"**

### 思维过程拆解

```
段永平的推理过程：

第一步：正面因素（为什么要买）
  生意模式不错（社交网络+广告）
  文化不错（扎克伯格年轻有为）
  价格合理

第二步：负面因素（为什么不舒服）
  自己不用FB的产品 → 缺乏第一手体验
  对FB的理解"没达到很透彻的地步" → 在能力圈边缘

第三步：触发事件
  抖音对FB构成威胁，FB反应慌乱 → "说明他可能还是有弱点"
  这个弱点是什么？是商业模式本身的问题，还是管理层应对问题？
  段永平不确定，但"不确定"本身就是退出的理由

第四步：自我诊断
  "对自己不够了解的公司，涨了也想卖跌了也想卖"
  → 这是一个段永平式的终极检验：如果你总是想卖，说明你没懂

第五步：决策
  卖出
```

### 关键洞察：不卖的原因可能比卖的原因更重要

段永平卖FB不是因为FB变差了，而是因为**他自己没有变懂**。

```
买入时：以为自己懂了（或"差不多懂了"）
持有中：发现自己其实没那么懂（不用产品、对竞争判断不确定）
卖出时：承认自己不懂，退出
```

**这是一个极其诚实的行为。** 大多数人不会承认"我不懂"，而是找一个理由："竞争加剧了"、"市场不好了"。

### 这对你分析公司的启示

```
问自己：如果我现在持有这个公司，涨了我想卖吗？跌了我想卖吗？
  
→ 如果答案都是"不想卖" → 你很可能真懂了
→ 如果答案是"涨了想卖跌了也想卖" → 你没懂（段永平的判断标准）
→ 如果答案是"涨了不想卖跌了想卖" → 你或许有投机心态
```

### 映射到分析框架

| 框架维度 | 应用 |
|---------|------|
| **能力圈** | 段永平在这里展示了能力圈的严格边界——甚至对自己买的公司也一样 |
| **排除测试** | 这是事后的排除：持有过程中发现自己并不真懂 → 退出 |
| **安全边际=理解度** | 段永平对FB的理解度不够 → 没有安全边际 → 应该卖出 |
| **诚实面对自己** | 这是段永平最值得学习的地方——"承认不懂"需要勇气 |

### 模仿要点——你分析公司时可以问的问题

> **"我用这个公司的产品吗？我真的理解它的用户体验吗？"**

如果答案是否定的，你可能也在能力圈之外。

## 案例六：GE——好公司犯错能回来

### 段永平的原话

> **"当GE掉到10块钱以下时（最低破6块了），我想GE好的时候能赚差不多两块钱，只要经济恢复正常，他怎么还不赚个一块多？由于ge的rating长期来讲还是会非常高的，假设他能赚1.5，给他个15倍pe不就20多了吗？我当时的想法就是这样的，简单吗？其实背后有一个不太简单的东西就是我能够坚信GE是家好公司（Great company），好公司犯错后能回来机会非常大。其实比我会算的人大把，但知道如何坚信确实不容易。"**

### 思维过程拆解

```
段永平的推理过程：

步骤1：判断GE是好公司（最核心的一步）
  "好公司犯错后能回来机会非常大"
  → 前提是确定它是好公司
  → 这不是分析出来的，是"坚信"

步骤2：毛估估正常利润
  "好的时候能赚差不多两块钱"
  "经济恢复正常后，保守估计1.5块/股"

步骤3：给合理PE
  考虑GE的历史评级和行业地位 → 15倍PE
  1.5 × 15 = 22.5

步骤4：和当前价格比
  6-10块 vs 22.5（毛估估）
  → 明显低估

步骤5：决策
  买入

关键洞察："其实比我会算的人大把"
  估值公式很简单（加法+PE），谁都会算。
  关键是：在价格从22跌到6的过程中，你还能不能"坚信"？
  绝大多数人在价格下跌时会怀疑自己的判断。
  段永平的"坚信"来自GE的历史和文化——他判断这是一家好公司。
```

### "坚信"的来源拆解

段永平的"坚信"不是盲目的信仰，而是基于：

```
1. 对GE商业模式的判断
  - 多元化工业集团
  - 有强大的品牌和行业地位
  - 有深厚的企业文化

2. 犯错的性质判断
  GE的问题是在"把事情做对"中犯错（金融业务的过度扩张）
  不是"做了错的事情"（如欺诈）
  所以可以恢复

3. 时间视角
  "好公司犯错后能回来机会非常大"
  短期有波动，长期优秀公司能自我修复

4. 价格因素
  因为犯错导致的价格暴跌提供了一个罕见的买入机会
```

### 映射到分析框架

| 框架维度 | 应用 |
|---------|------|
| **前置筛选** | GE在正常情况下能通过排除测试吗？可以（好生意模式+好文化） |
| **犯错性质** | 区分了"做错的事"和"把事情做对中的错"——这是段永平极其重要的判断工具 |
| **商业模式** | GE当时的商业模式仍被段永平认可（虽然后来证明GE的问题更深） |
| **价格** | 暴跌提供了难得的安全边际（或更准确地说，理解度×低价=机会） |
| **反事实检验** | 价格从22跌到6，段永平想的是"买"，不是"跑" → 说明他真懂（或他以为自己懂） |

### 历史注脚：段永平这次错了吗？

GE后来并没有恢复——它的商业模式其实被段永平误判了（金融业务的系统性风险比段永平理解的大）。但这正好说明了段永平自己说的另外两句话：

> **"错误是不可避免的，但呆在能力圈内以及专注和用功可以大幅度减少犯错的机会。"**

> **"发现买错了股票或公司就应该赶紧离开，不然越到后面损失会越大。"**

段永平在GE上的经历反而印证了他自己的原则：**即使是最优秀的投资者也会犯错，重要的是承认错误并及时退出。**

### 模仿要点——你分析公司时可以问的问题

> **"这个公司当前的问题是'做了错的事'还是'做事中犯了错'？"**

这个判断决定了：是应该买入（好公司犯错可以修复），还是应该远离（根本性错误无法修复）。

## 案例七：雅虎——毛估估的典范

### 段永平的原话

> **"每股（都是大概数，没细算过）：现金3，yahoo以外上市部分属于yahoo的部分（包括yahoo Japan约30%和阿里巴巴B2B香港上市部分约30%）4.6+2.1=6.7。yahoo本身现在的现金流大概有每股1.4/年，估计盈利大概在0.8左右，在Internet这样一个成长的市场里，yahoo的广告总量还是有很大可能成长的，所以我给这部分12倍的pe，这就是9.6。以上总和是3+6.7+9.6=19.3。所以我认为yahoo现在的市场的fair price大概就在18-20块之间，所以我认为yahoo不贵。"**

### 思维过程拆解

```
步骤1：分拆资产
  雅虎不是一个整体，是三个部分：
  A = 净现金（每股3）
  B = 持有的其他公司价值（每股6.7）
  C = 主营业务（每股盈利0.8）
  → 分别估值

步骤2：先估最容易的部分
  净现金：3/股 → 确定性最高
  持有资产：6.7/股 → 基于市场价格，确定性也高

步骤3：估主流业务
  盈利0.8/股
  给12倍PE → 9.6/股
  这里用了"保守估计"（盈利大概在0.8左右）

步骤4：加总
  3 + 6.7 + 9.6 = 19.3

步骤5：和市价比较
  → 不贵（小于或等于20块）
```

### 这个案例展示的毛估估原则

| 原则 | 体现 |
|------|------|
| **分拆估值** | 把复杂的公司拆成简单的部分，分别估值 |
| **先估确定性高的部分** | 现金→持股→业务的顺序 |
| **保守估计** | "盈利大概在0.8左右"——偏保守 |
| **用范围代替精确** | "fair price大概就在18-20块之间"——不是19.3精确值 |
| **简单数学** | 没有DCF、没有参数、没有公式 |

### 模仿要点——你分析公司时可以问的问题

> **"如果我把这个公司拆成几块，每块值多少钱？"**

这是最实用的毛估估方法。很多公司看着复杂，拆成"现金+投资资产+主营业务"就清晰了。

## 案例八：拼多多——流量+本分文化（尚未验证）

### 段永平的原话

> **"我懂得不多，但知道流量经常是很值钱的。他们流量很大，而且都来源于看得见的生意，所以变现不是问题。我相信他们本分的文化会让他们找到非急功近利的办法的。"**

### 思维过程拆解

```
段永平的态度：谨慎正面

支持因素：
  1. 流量大 → 值钱（他理解流量的价值）
  2. 流量来源清晰 → 不是刷出来的
  3. 本分文化 → 黄峥是他的门徒，他认可黄峥的理念

不确定因素：
  1. "我懂得不多" → 坦诚自己在电商领域不是专家
  2. "相信" → 这是信任判断，不是商业判断
  3. 拼多多还在发展早期 → 商业模式未完全验证
```

这个案例显示了段永平在**能力圈边缘**时的态度：相信但保持谨慎，投了但比例不大。

## 案例十：阿里巴巴——企业文化的价值

> **"阿里巴巴的企业文化是我在中国企业里迄今见到的写的最好的企业文化。拥有这么好的企业文化再加上已经找到的这么好的生意模式，阿里巴巴想不成功都不容易啊。这也是我愿意买Yahoo的最重要的原因！"**

段永平的逻辑链条：

第一步：看企业文化（这是阿里巴巴最突出的点）

## 案例十一：泡泡玛特——迭代认知的完整示范

### 背景

段永平从2025年到2026年经历了完整的态度转变：

| 时间 | 段永平的态度 | 触发因素 |
|------|-------------|---------|
| 2025年8月 | "看不懂10年后公司会怎样" | 第一次接触泡泡玛特 |
| 2025年12月 | "不会买泡泡玛特的股票" | 与王石对话时的表态 |
| 2026年1月 | "无法理解人们为什么会需要这个东西" | 仍然看不懂 |
| 2026年3月30日 | **态度转折**："决定收回'不投资'的说法" | 2025年年报发布（营收+184%，净利130亿） |
| 2026年4月 | 读《因为独特》→ 实地考察伦敦门店 | 开始理解王宁 |
| 2026年5月7日 | "把神华都换了泡泡玛特" | 清仓持有十几年的神华，ALL IN |
| 2026年5月25日 | "王宁对商业的理解比乔布斯还强一点点" | 成为第二大股东，持有118亿港元 |

### 段永平的原话（按时间顺序）

> **"看不懂10年后公司会怎样。"**（2025年8月）
>
> **"也许已经过了能理解这个产品的年纪了。"**（2025年12月）
>
> **"无法理解人们为什么会需要这个东西，万一过两年大家都不要了呢？"**（2026年1月）
>
> **"这两天花时间再看了看泡泡玛特，决定收回之前不投资的说法。"**（2026年3月30日）
>
> **"不大的门店生意极好，进店顾客绝大多数是本地成年人，华人占比仅一成左右。我怎么觉得泡泡玛特才是中国产品国际化的先驱呢？"**（2026年4月）
>
> **"我把我的神华都换了泡泡玛特了。"**（2026年5月7日）
>
> **"我觉得王宁对自己产品的理解以及追求和Jobs是一个级别的……王宁对商业的理解好像比Jobs还要强一点点。"**（2026年5月25日）
>
> **"他还能至少好好干25年以上。这个复利是吓人的。"**（2026年5月）

### 思维过程拆解（迭代认知循环）

段永平的决策不是一次性"看懂→买入"，而是经历了一个**完整的迭代认知循环**：

```
第一轮：基于现有框架的判断（2025年8月）

输入信息：
  泡泡玛特是卖盲盒的公司
  盲盒=潮玩，潮玩=可能一阵风
  不理解人们为什么需要

推理过程：
  用自己的经验框架套：电子宠物、呼啦圈、魔方 → 一阵风
  结论：看不懂 → 不碰

⚠️ 问题：段永平用自己的"老经验"套了新东西
  这在大部分时候有效（大多数流行确实是一阵风）
  但他忽略了泡泡玛特可能不只是"盲盒"
```

```
第二轮：反直觉信号触发（2026年3月）

输入信息：
  年报数据：营收371亿（+184%），净利130亿（+284%）
  毛利率72%，净利率34%

推理过程：
  "居然真的有这么好的盈利"
  一个"卖玩具的"怎么可能有这么好的数据？
  → 触发"异常信号"：要么我错了，要么这数据不可持续
  → 决定重新审视

⚠️ 关键点：他不是直接否定数据，而是怀疑自己的判断
  这是迭代认知的核心——"异常信号"不是终点，是重新研究的起点
```

```
第三轮：创始人深潜（2026年4月）

行动：
  1. 读完王宁传记《因为独特》
  2. 亲自到伦敦门店考察
  3. 观察用户画像："本地成年人，华人仅一成"

推理过程：
  读传记后："我能看懂他有多厉害"（企业家共情）
    → 王宁不是"卖盲盒的"，他是"做IP平台"的
    → 他对产品和商业的理解是顶级的
  实地考察后：不是中国人炒作，不是小孩的东西
    → 国际化是真的，成年人消费是真的
    → IP的陪伴价值是存在的

类比突破："泡泡玛特=游戏化的服装公司"
    → 他自己玩《梦幻西游》买皮肤的经历
    → 泡泡玛特卖的不是玩具，是"情绪价值"和"陪伴"
    → 这和游戏皮肤、茅台酒、苹果手机一样，都是心理需求
```

```
第四轮：重新定价（2026年4-5月）

新的认知框架：
  泡泡玛特不是"盲盒公司"
  → 是"IP平台公司"，类似迪士尼的逻辑
  → 高毛利（72%）+高净利（34%）+轻资产+全球化
  → 创始人能再干25年以上

重新估值：
  之前：无法估值（不在能力圈内）
  现在：PE 11-13倍（理解后，这个价格显得便宜）

决策：
  "把神华都换了泡泡玛特"——118亿港元，成第二大股东
```

### "异常信号触发"在这个案例中的完整应用

这个案例完美展示了`05-估值与决策.md`中的"异常信号触发机制"：

```
1. 反直觉信号出现
   "一个卖盲盒的公司，净利率34%，净利130亿？"
   
2. 触发"估值重置"
   从"不估值"切换到"重新审视"
   
3. 验证信号的真实性
   读传记（验证创始人的质量）
   实地考察（验证海外数据的真实性）
   
4. 修正认知框架
   从"盲盒公司"→"IP平台公司"
   从"一阵风"→"情绪消费的长期趋势"
   
5. 重新估值
   从"无法估值"→"PE 11-13倍，便宜"
```

### 这个案例对照明框架缺失的启示

| 缺失 | 在泡泡玛特案例中的体现 | 我们在框架中补了什么 |
|------|---------------------|------------------|
| **创始人深潜** | 理解了王宁才能理解泡泡玛特的生意 | 新增独立创始人深潜维度和方法论 |
| **实地验证** | 蹲在伦敦店门口数人头，确认本地成年人在买 | 新增实地验证步骤 |
| **迭代认知** | 9个月经历了4轮认知迭代 | 新增迭代认知循环和异常信号触发机制 |
| **双路径模型** | 泡泡玛特是B型（创始人优先），不能用A型框架套 | 新增公司类型判断Step 0和双路径 |
| **企业家复利** | "他还能至少好好干25年以上" | 新增企业家复利概念 |
| **类比思考** | "泡泡玛特=游戏化的服装公司" | 新增类比突破在产品理解中的位置 |

### 模仿要点——你分析公司时可以问的问题

> **"这个公司我9个月前看不懂，现在我能看懂了吗？"**

如果答案一直是"看不懂"，可能确实不在能力圈内。
如果答案变成了"现在能看懂了"，说明认知迭代了——**关键是要知道是什么信息改变了你的判断。**

> **"如果我今天清仓了这家公司，什么信息会让我反悔？"**

这个问题的反面是：你现在有什么信息是别人不知道的？如果没有，你可能并没有"真懂"。
段永平有：他读了王宁的传记，他蹲在门店数人头，他有企业家的视角。

> **"这个公司的生意，能不能用我熟悉的一个东西来类比？"**

段永平用"游戏皮肤"理解了泡泡玛特。**类比是连接新事物和已知经验的最强工具。**
如果找不到类比，可能还没真正理解这个生意。
