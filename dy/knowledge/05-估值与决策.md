# 段永平论估值——毛估估与决策框架

> 以下内容均整理自段永平原话，来源为《段永平投资问答录（投资逻辑篇）》及网易博客。
>
> **本文用于分析流程 Step 3（价格评估）和 Step 4（综合判断）。**
> 段永平对估值的核心态度是：**估值不是算出来的，而是"比较"出来的。** 不要试图精确计算一个公司的价值，而是问自己：这个生意未来10年能赚多少钱？比我其他的选择好吗？

---

## 一、根本思路：未来现金流折现（DCF）

> **"买股票就是买公司，买公司就是买其未来现金流(的折现）。这里现金流指的是净现金流，未来指的是公司的整个生命周期，不是3年，也不是5年。"**

### 这不是公式，是思维方式

> **"未来现金流折现只是一种思维方式，千万不要去套公式，因为没人可以真的确定公式中的变量，所有假设可能都是不靠谱的。"**

段永平反复强调这一点，因为他知道**精确的错误比模糊的正确更危险**。

### 正确用法

```
不正确的做法：
  预测第1年现金流100亿，第2年110亿...第10年259亿 → 算8%折现率 → 得某个精确数字 → "价值1500亿！"
  （这就是"套公式"）

正确的做法：
  这个生意10年后大概比现在赚得多还是少？多很多还是一点点？          → 方向判断
  以现在的价格买入，10年后我的年化回报大概是多少？比国债好还是差？   → 比较判断
  我对这个判断的确定性有多高？                                      → 安全边际判断
```

### 价格在决策框架中的位置

```
┌─────────────────────────────────────┐
│ 第三关：价格/估值 ← 你在这里         │
│ 权重：25%（三个维度中最低）           │
│ 条件：仅在前两关（模式+文化）通过后进入│
├─────────────────────────────────────┤
│ 核心问题：现在这个价格，值得买吗？     │
│ 好生意+好文化+好价格 = 下重注机会      │
│ 好生意+好文化+合理价格 = 可以买       │
│ 好生意+好文化+太贵了 = 等            │
└─────────────────────────────────────┘
```

## 二、折现率 = 机会成本（核心概念）

> **"折现率实际上是相对于投资人的机会成本而言的。最低的机会成本就是无风险回报率，比如美国国债的利率。"**

> **"实际的最小机会成本应该是通胀率。但通胀率其实不是机会成本而应该是'基本回报率'，在投资时其实无法考虑通胀率，但可以比较利率。也就是说，最小机会成本还是无风险国债回报。"**

### 机会成本的层次结构

```
你的资金只有一笔，做了A就不能做B。

买这个公司
    vs
你能力圈内最好的公司（如段永平的茅台、苹果）
    vs
指数基金（市场平均回报）
    vs
国债/存款（无风险回报）

→ 选10年后回报最高的那个
```

> **"我其实并不在乎别人这一时期赚了多少钱，我只考虑自己的机会成本。"**

### 机会成本的实际操作

段永平的做法：

> **"A股里我从若干年前就基本只拿着茅台，脑子里总是想着10年后茅台会怎么样，所以也容易拿得住，回报也不错。我并不在乎别人这一时期赚了多少钱，我只考虑自己的机会成本。"**

**他的机会成本在A股是"茅台"，在美股是"苹果"。** 当一个新公司进入视野时，他的问题是：**这比茅台/苹果好吗？**

## 三、PE的正确用法

### PE是历史数据

> **"PE是历史数据。我们买的是未来的现金流。所以除非你能从历史中看到未来，否则没意义。"**

> **"市盈率是倒后镜，不能作为能不能投的标准。关键是未来的盈利能力，看不懂的再低也别碰。"**

### PE的参考价值

段永平虽然认为PE是"倒后镜"，但他自己也在用PE做参考：

| 用法 | 原话 | 含义 |
|------|------|------|
| **和未来利润结合** | "我说的pe一般指的是相对于未来长期实际利润的pe" | 使用的不是静态PE，而是基于未来利润的"调整PE" |
| **作为粗略参考** | "多少倍pe和平均长期利息有关，12倍长期pe应该可以了" | 12倍≈8.3%回报率，和无风险利率大致匹配 |
| **看净现金调整后的PE** | "扣掉现金的话，苹果的今年的未来pe只有12-13倍啊" | 调整后的真实PE更有意义 |
| **警惕低PE陷阱** | "pe很低的公司很可能是有问题的公司" | 低PE可能反映市场看到了你不知道的风险 |

### 段永平的PE速查表

```
PE = 10  → 10年回本（如果利润不增长）
PE = 12  → "长期pe应该可以了"
PE = 15  → 美国股市长期平均
PE = 20+ → 需要高增长来支撑
PE = 50+ → 新股常见，但"可真是需要勇气啊"
```

### PE的进阶使用——e/e

> **"相对于pe，我会更多用e/e来衡量一个企业。"**

e/e：**当前股价相对于企业长期平均盈利能力的倍数**。即：
- 不是看今年的利润，而是看"正常情况下"这个公司能赚多少
- 忽略短期波动，看长期平均盈利能力
- 对周期性公司尤其适用

### 诊断问题

```
□ 当前PE是基于历史利润还是未来利润？
□ 如果调整了净现金，真实PE是多少？
□ PE低是因为真便宜，还是因为有潜在风险？
□ 公司未来3-5年的利润趋势是往上还是往下？
□ 用e/e（长期平均利润）看，PE是多少？
```

## 四、ROE的正确用法

> **"我也喜欢ROE这个指标。一般来说，ROE高的公司商业模式好。"**

> **"Roe或Roa在了解一个公司的生意模式时是有帮助的，但不是充分条件。"**

### ROE诊断流程

段永平给出了一个清晰的ROE分析逻辑：

```
ROE高 → 引人注目
    ↓
看债务
    ├── 有高负债 → 高ROE可能是杠杆的结果，不是生意真好
    │              例：银行ROE通常高，但因为有高杠杆
    │
    └── 无高负债 → 高ROE反映了真实的盈利能力
                      ↓
                 看文化 → 这种高盈利能持续吗？
                      ↓
                 回到 "right business, right people, right price"
```

> **"'净资产收益率'是个很好的指标，可以用来排除那些你不喜欢的公司，但不能作为'核心指标'去决定你是否要投的公司。这个指标如果好的话，你首先要看的是债务；如果没有债务问题则马上要看文化，看这种获利是否可以持续。最后还是要回到'right business, right people, right price'。"**

### 诊断问题

```
□ ROE高是因为生意好还是因为负债高？
□ 如果去掉了杠杆，ROA（资产回报率）是多少？
□ 高ROE能持续吗？受什么因素影响？
□ ROE的趋势——是在改善还是恶化？
```

## 五、安全边际 = 理解度，不是价格

这是段永平对传统"安全边际"概念最重要的修正。

> **"我认为老巴的'安全边际'实际上指的是理解度而不是价格。从10年的角度看，买的时候比最低价贵30%其实没多少。"**

> **"安全边际应该指的就是你自己的了解度，投资里没有数学上100%的东西。"**

> **"如果指的是你对10年后公司的理解，那把握度就是你的安全边际。"**

### 段永平的"安全边际" vs 传统"安全边际"

| 维度 | 传统理解 | 段永平的理解 |
|------|---------|-------------|
| 定义 | 价格低于内在价值的差额 | **你对公司的理解度** |
| 获取方式 | 计算内在价值，找到低估 | 深入研究，提高理解度 |
| 保护来源 | 价格折扣 | 确定的认知 |
| 数量化 | 可以（如"低于内在价值30%"） | 不可精确量化（"毛估估"） |
| 如何增加 | 等更便宜的价格 | **花时间搞懂公司** |

### 如何增加安全边际（按段永平的理解）

```
方法一：等待更低价格 → "好公司也需要好价格"
  （但作用有限——"比最低价贵30%其实没多少"）

方法二：加深理解 → 这才是真正的安全边际
  读年报、了解产品、跟踪管理层言行、
  对比竞争对手、站在消费者角度体验产品

方法三：只在能力圈内操作
  "呆在能力圈内以及专注和用功可以大幅度减少犯错的机会"
```

> **"对一个伟大公司而言，有时候一点点价格差别10年后看都不是事，但因为差那点价格而错过一个好公司就可能是大事了。"**

## 六、买卖决策框架

### 什么时候买

> **"真正看懂的公司价格合适就能买。"**

条件必须同时满足：
```
☑ 看懂了（在能力圈内，能说清10年后的现金流方向）
☑ 价格合适（相对于机会成本而言合理或便宜）
```

#### "价格太贵"怎么办

> **"有些公司的生意模式很好，但股价有时候太贵，那就只能等了。好在这些年来一直如此，每隔些年就来个股灾，往往那时好公司也会跟着稀里哗啦的。"**

段永平的策略是**等待**——好公司总会有打折的时候（股灾、行业危机、短期利空）。

> **"从10年的角度看，买的时候比最低价贵30%其实没多少。"**

价格容忍度参考：

```
价格水平               段永平的行为
极度低估（股灾时）  →  下重注（"好公司犯错后能回来"）
合理偏低            →  可以买，加大仓位
合理                →  可以买
合理偏高（贵10-30%）→  可以买（如果确定性极高）
太贵（贵50%以上）   →  等
```

### 什么时候卖

段永平讨论过三种卖出的情况：

**情况一：发现买错了**

> **"发现买错了股票或公司就应该赶紧离开，不然越到后面损失会越大。"**

这是最简单的卖出理由——**承认错误**。段永平强调要区分"做错的事"和"把事情做对中的错误"：

> **"如果懂沉没成本以及机会成本的概念的话就会容易很多。"**

**情况二：发现更好的机会（机会成本变化）**

隐含在机会成本框架中——如果出现了明显更好的投资机会，可以换仓。

**情况三：对公司的理解发生变化**

> **"FB准备卖掉了，有些不舒服的东西。由于自己不用FB的任何产品，所以对FB的理解一直没达到很透彻的地步。"**

> **"对自己不够了解的公司，涨了也想卖跌了也想卖。"**

**卖出决策的段永平式检查清单：**
```
□ 公司基本面变了吗？还是只是价格变了？
  基本面没变 → 不卖（"好生意好文化，拿着能睡好觉"）
  基本面变了（模式或文化出问题）→ 卖

□ 如果是价格跌了：我想买更多还是想跑？
  想买更多 → 真懂，不卖
  想跑     → 可能没真懂（或者基本面的确变了）

□ 如果是价格涨了：现在卖的话，我还能用什么更好的选择替代它？
  没有更好的选择 → 不卖
  有更好的选择 → 考虑换仓（机会成本）
```

### 关于持有

> **"脑子里总是想着10年后茅台会怎么样，所以也容易拿得住。"**

> **"好的生意模式和企业文化不错的公司，价钱合适，拿着能睡好觉。"**

**持有的关键：** 不是"要不要卖"，而是"10年后这家公司会怎样"。如果能清楚回答这个问题，持有就不是问题。

## 七、段永平的"毛估估"示范

### 案例一：雅虎

> **"每股（都是大概数，没细算过）：现金3，yahoo以外上市部分属于yahoo的部分（包括yahoo Japan约30%和阿里巴巴B2B香港上市部分约30%）4.6+2.1=6.7。yahoo本身现在的现金流大概有每股1.4/年，估计盈利大概在0.8左右，在Internet这样一个成长的市场里，yahoo的广告总量还是有很大可能成长的，所以我给这部分12倍的pe，这就是9.6。以上总和是3+6.7+9.6=19.3。所以我认为yahoo现在的市场的fair price大概就在18-20块之间，所以我认为yahoo不贵。"**

**分析过程拆解：**

```
步骤1：计算净现金价值
  每股现金 3.0

步骤2：计算持有其他公司的价值
  yahoo Japan 30% → 4.6
  阿里巴巴B2B 30% → 2.1
  合计 6.7

步骤3：计算主营业务的估值
  正常盈利 0.8/股
  给12倍PE → 9.6/股

步骤4：加总
  3.0 + 6.7 + 9.6 = 19.3

步骤5：判断
  当前价低于19.3 → 不贵
```

注意：段永平没有用复杂的DCF模型，就是**简单的资产加总 + 合理PE**。这就是"毛估估"。

### 案例二：GE

> **"当GE掉到10块钱以下时（最低破6块了），我想GE好的时候能赚差不多两块钱，只要经济恢复正常，他怎么还不赚个一块多？由于ge的rating长期来讲还是会非常高的，假设他能赚1.5，给他个15倍pe不就20多了吗？"**

**分析过程：**

```
步骤1：确定正常盈利水平
  好的时候赚2块/股
  经济恢复后保守估计1.5块/股

步骤2：给合理PE
  考虑GE的评级和历史地位 → 15倍PE

步骤3：得出估值
  1.5 × 15 = 22.5

步骤4：和当前价（6-10块）对比
  远低于估值 → 买入
```

**关键：背后有一个"不太简单的东西"——**

> **"其实背后有一个不太简单的东西就是我能够坚信GE是家好公司（Great company），好公司犯错后能回来机会非常大。"**

**毛估估的价值不在于精确，而在于你对公司的信念。**

### "毛估估"的原则总结

```
1. 不要试图精确    → 用范围而不是具体数字
2. 用简单逻辑      → 加法+合理PE就够了
3. 保守            → 利润估计偏保守，PE给合理值
4. 背后的确定性最重要 → 理解越深，越敢"毛估估"
5. 和机会成本比较 → 比国债好、比茅台好、比苹果好？
```

## 八、常见估值误区

### 误区一：追求精确

> **"没人真用公式的，至少剩者都是不用公式的。（说自己有公式还有参数的都是不懂甚至可能是骗人的，别理他们）"**

### 误区二：静态看PE

> **"千万不要被pe误导，因为pe是个历史数据，未必能说明未来。"**

### 误区三：低PE就是便宜

> **"不能简单的单纯看数字，除非账面净现金多过股价了。"**

### 误区四：高成长就值高PE

> **"对高成长的企业来说，pe高些是可以接受的，只要你能确定他的高成长就行。如果你明白未来现金流折现的概念的话，就不会有这个问题。"**

**关键在"能确定"三个字**——大部分高成长公司你无法确定。

### 误区五：只看公司本身，不看机会成本

> **"我其实并不在乎别人这一时期赚了多少钱，我只考虑自己的机会成本。"**

**估值永远是相对的不是绝对的。**

## 九、估值确定性等级判断

| 等级 | 含义 | 行动 |
|------|------|------|
| **明确低估** | 价格显著低于毛估估价值，且确定性高 | 可以买，甚至可以下重注 |
| **合理偏低** | 价格略低于或等于毛估估价值 | 可以买 |
| **合理** | 价格与价值匹配 | 可以买，但预期回报合理 |
| **合理偏高** | 价格略高于毛估估价值 | 如果确定性极高（模式+文化都顶级），仍然可以买 |
| **太贵** | 价格显著高于毛估估价值 | 等——"每隔些年就来个股灾" |
| **看不懂** | 无法判断价值和价格的关系 | "不懂不做" |

---

> **"我其实并不懂通常意义上的估值，就是所谓股价应该是多少的那种估值。"**
>
> ——段永平说这句话的意思是：**真正的估值不是算一个目标价，而是判断"比我的其他选择更好"**。

## 十、迭代认知循环（新增）

### 10.1 为什么估值需要迭代？

你的估值是基于**当前的认知**做出的。根据`01-认知金字塔-总纲.md`的双路径模型，**认知是可以升级的**。

段永平对泡泡玛特估值的进化：

```
2025年8月：看不懂 → 不估值（不在能力圈内）

2026年3月（年报后）：
  营收371亿，净利130亿 → "居然真的有这么好的盈利"
  → 开始思考估值，但不急于下结论

2026年4月（传记+实地考察后）：
  理解了创始人 → 重新审视生意的可持续性
  → 估值框架变化：从"看不懂"到"可以用PE毛估估"

2026年5月（全面建仓）：
  PE 11-13倍 → "right price"
  → 在理解生意后，价格显得便宜
```

**核心洞察：不是泡泡玛特的价格变了，是段永平的"能力圈"扩大了。**

### 10.2 异常信号触发机制

当遇到**反直觉的信号**时，应该触发"估值重置"：

**什么是反直觉信号？**

| 类型 | 例子 | 应该问自己 |
|------|------|-----------|
| 数据好到不相信 | "一个盲盒公司居然能赚130亿？" | 如果这是真的，我的哪些假设是错的？ |
| 数据坏到不相信 | "好公司突然暴雷？" | 这个坏消息是一次性的还是结构性的？ |
| 反复出现的同类信号 | 不同来源都说同一件事 | 我是否错过了什么重要的变化？ |
| 创始人行为异常 | 创始人大量套现 | 他看到了什么我没看到的？ |

**触发后的行动：**

```
1. 停止执行原决策
   如果你原本打算买入/持有/卖出 → 先暂停

2. 问自己：这个信号如果是真的，我的判断哪里错了？
   这是"验证假设"而不是"坚持原有判断"

3. 去验证信号的真实性
   是数据错了还是我理解错了？
   数据：查原始来源，看是否统计口径变化
   理解：是否有新的信息改变了对生意模式的判断

4. 如果信号为真 → 修正认知 → 重新估值
5. 如果信号为伪 → 坚持原有判断 → 继续持有
```

### 10.3 两种估值修正模式

| 模式 | 含义 | 触发条件 | 行动 |
|------|------|---------|------|
| **小修** | 现有估值框架不变，修正参数 | 常规财报波动 | 微调利润预期，PE倍数不变 |
| **大修** | 估值框架本身需要重构 | 反直觉信号出现 | 重新检查商业模式和文化假设，可能需要重新走一遍分析流程 |

**段永平的泡泡玛特案例是大修**——他从"不估值"模式切换到"PE估值"模式，不是调整了数字，是**框架本身变了**。

### 10.4 "能力圈扩张" vs "股价变化"

一个关键区分：

```
股价下跌 ≠ 公司变便宜了（如果你还不懂）
  → 只有你懂了，价格才有意义
  
股价上涨 ≠ 公司变贵了（如果你更懂了）
  → 认知升级可能导致你愿意给更高估值
```

段永平在泡泡玛特的实践：
- 从"看不懂"（不估值）→ "看懂了"（PE 11-13倍是便宜）
- **是能力圈扩张带来了估值框架的变化，不是股价变化**

### 10.5 迭代式估值检查清单

```
当前我的估值是基于什么假设？
  □ 对公司生意模式的理解：高/中/低
  □ 对公司创始人的理解：高/中/低（B型公司专用）
  □ 对未来3-5年利润的方向判断：向上/持平/向下
  
我的估值需要关注哪些信号？
  □ 下一个关键信号是什么？（财报/产品/海外数据）
  □ 什么数据会让我上调估值？
  □ 什么数据会让我下调估值？
  
如果我有机会重新分析一遍（回到Step 0）：
  □ 我的结论会变吗？
  □ 哪些信息让我改变了想法？
```

### 10.6 迭代思维 vs 传统估值思维

| 维度 | 传统估值 | 迭代式估值 |
|------|---------|-----------|
| 时间 | 一次估值，定期刷新 | 持续跟踪，等待认知突破 |
| 认知 | 假设认知不变，只更新数据 | 假设认知可以升级，主动寻求突破 |
| 价格容忍度 | 给定估值框架下的容忍度 | 认知升级后可能重新定价 |
| 适用于 | A型公司（生意自明） | B型公司（需要深度理解） |
| 核心问题 | "当前价格便宜吗？" | "我比以前更懂了吗？" |

> **段永平说："看懂一家公司不会比读一个本科更容易。"**
>
> 既然要花3-4年读完一个本科，那花9个月理解一个公司也没什么奇怪。
> **给自己时间迭代认知，比追求一次看懂更现实。**
