## 🚀 使用方式

### 方式一：综合分析一家公司（推荐）

直接在 Claude 中说：

> **"用段永平的方法综合分析一下 [公司名]"**

Claude 会自动按照 `dy/knowledge/01-认知金字塔-总纲.md` 中的**四层九步分析流程**进行：

```
Step 0: 环境变化检测（⭐⭐⭐ 新增——最先做）
   颠覆性信号扫描：技术变革？消费者行为变化？竞争格局变化？
   段永平历史判断的前提条件检查
   结构性颠覆 → 不硬套框架，走"框架外思考"路径

Step 0.5: 公司类型判断 → 生意优先型(A)还是创始人优先型(B)？

Path A（生意优先——茅台、苹果）：
  排除测试 → 生意叙事 → 商业模式(40%)→ 文化(35%)→ 价格(25%)

Path B（创始人优先——泡泡玛特）：
  排除测试 → 生意叙事 → 创始人深潜 → 商业模式(35%)→ 文化(35%)→ 价格(30%)

通用收尾：实地验证 → 综合判断 + 10年视角 → 迭代认知循环
```

### 方式二：快速筛选（排除测试模式）

> **"用段永平的方法快速帮我筛选一下 [公司名]，主要看有没有一票否决的问题"**

只做排除测试，大部分公司在这个阶段就会被跳过——这本身就是段永平的核心方法。

### 方式三：针对特定维度提问

> - "段永平会怎么看 [公司] 的商业模式？" → 参考 `dy/knowledge/03-商业模式分析框架.md`
> - "[公司] 的企业文化够本分吗？" → 参考 `dy/knowledge/04-企业文化诊断框架.md`
> - "用段永平的方法估一下 [公司] 值多少钱？" → 参考 `dy/knowledge/05-估值与决策.md`
> - "段永平当年是怎么分析 [苹果/网易/腾讯] 的？" → 参考 `dy/knowledge/06-案例思维拆解.md`

### 方式四：学习段永平的思维方式

> **"段永平是怎么思考投资的？"**

从 `dy/knowledge/01-认知金字塔-总纲.md` 的总纲开始，然后：
- 读 `dy/knowledge/02-核心原语.md` 理解基础原则
- 读 `dy/knowledge/06-案例思维拆解.md` 看这些原则在具体案例中的运用

## ⚠️ 使用原则

1. **所有引用的段永平观点必须出自 `dy/knowledge/` 目录下的原文**
2. **不添加任何非段永平本人的判断作为"段永平的观点"**
3. **分析报告中的个人推论需与段永平原话明确区分**
4. **① 先做环境变化检测（Step 0），再进入任何分析——检查段永平过去判断的前提条件还在不在**
5. **② 结构性颠覆时不要硬套框架——诚实说"我不知道"比"用框架给出结论"更有价值**
6. **③ 再判断公司类型（A型还是B型），选分析路径**
7. **④ 创始人驱动的公司（B型）从创始人深潜开始，不要套用标准清单**
8. **⑤ 权重体系是思维辅助工具，不是数学公式**
9. **⑥ 给自己时间迭代认知——月度/季度复盘，更新判断**

> **"其实我对投资的理解就这些，其他都是这几句话衍生出来的。"**
> ——段永平
>
> ⭐ **v3.0 更新说明**：本次重构新增了**环境变化检测层**（Step 0），要求所有引用段永平过去判断的分析必须先检查前提条件是否变化。解决的核心问题是：**"过去的判断在现在的环境下还成立吗？"** 当检测到颠覆性变化（如AI对社交平台的冲击）时，传统框架可能不适用，需要走"框架外思考"路径。详情见 `01-认知金字塔-总纲.md` 第八章。
